    <h1>Sorting Algorithm Visualizer</h1>
    <p>A Java Swing based project for visual learning</p>
</header>

<section>
    <h2>Project Overview</h2>
    <p>
        Sorting Algorithm Visualizer is a desktop-based application built using Java Swing.
        It helps students understand how sorting algorithms work through live animations,
        color indicators, and real-time statistics.
    </p>

    
        This project is designed for learning, practice, and academic presentations.
    
</section>

<section>
    <h2>Features</h2>
    <ul>
        <li>Real-time animation of sorting algorithms</li>
        <li>Comparison and swap counters</li>
        <li>Pause and resume functionality</li>
        <li>Adjustable animation speed</li>
        <li>Two display modes: Node view and Bar view</li>
        <li>Manual input support</li>
        <li>Random array generation</li>
        <li>Clean dark themed interface</li>
    </ul>
</section>

<section>
    <h2>Supported Algorithms</h2>
    <ul>
        <li>Merge Sort</li>
        <li>Quick Sort</li>
        <li>Insertion Sort</li>
        <li>Selection Sort</li>
        <li>Bubble Sort</li>
    </ul>

    Each algorithm highlights:
    
        Current comparison
        Active pointers
        Pivot or key
        Sorted elements
    
</section>

<section>
    <h2>Technologies Used</h2>
    <ul>
        <li>Java</li>
        <li>Java Swing</li>
        <li>AWT Graphics</li>
        <li>Multithreading</li>
    </ul>
</section>

<section>
    <h2>How the Project Works</h2>
    <div class="box">
        <ul>
            <li>Each number is represented as a visual node.</li>
            <li>Nodes move smoothly using position updates.</li>
            <li>Sorting runs on a separate thread to keep the UI responsive.</li>
            <li>Colors represent comparing, swapping, and sorted states.</li>
            <li>Statistics update in real time during execution.</li>
        </ul>
    </div>
</section>

<section>
    <h2>Controls Guide</h2>

    <ul>
        <li><b>Algorithm Selector:</b> Choose the sorting algorithm</li>
        <li><b>Speed Slider:</b> Control animation speed</li>
        <li><b>Play:</b> Start sorting</li>
        <li><b>Pause / Resume:</b> Pause or continue sorting</li>
        <li><b>Reset:</b> Clear and reset visualization</li>
        <li><b>Input Field:</b> Add custom numbers</li>
        <li><b>Randomize:</b> Generate random values</li>
        <li><b>View Toggle:</b> Switch between node and bar view</li>
    </ul>
